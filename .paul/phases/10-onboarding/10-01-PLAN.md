<objective>
## Goal
Show a 3-step onboarding overlay the first time a rider uses the app — triggered immediately after they choose to log in or continue as guest. Skippable, never shown again after completion.

## Purpose
First-time riders don't know what to do when they land in the calculator. A brief, focused onboarding removes that friction — 3 steps, skippable, designed for mobile, covering what the app does, how to use it, and what they get from an account.

## Output
- `#onboarding-overlay` — full-screen 3-step overlay with progress dots, skip button, and next/done CTA
- `showOnboarding()` / `nextOnboardingStep()` / `completeOnboarding()` JS functions
- Triggered from: successful auth login (first time) + `enterGuestMode()` (already has `showOnboarding()` call from Phase 9)
- Gated by `localStorage.getItem('rr_onboarded')` — never shown twice
</objective>

    **Step 1 — Add onboarding JS functions** near `enterGuestMode()` (end of Phase 9 additions):

    ```js
    // ── Onboarding ────────────────────────────────────────────────────────
    let onbCurrentStep = 1;
    const ONB_TOTAL = 3;

    function showOnboarding() {
      if (localStorage.getItem('rr_onboarded')) return;
      onbCurrentStep = 1;
      // Update step 3 body based on guest/logged-in state
      const step3body = document.getElementById('onb-step-3-body');
      if (step3body) {
        step3body.textContent = isGuest
          ? 'Sign up free to save your setups and access them anywhere — even at the trailhead.'
          : 'Your setups are saved to your account — access them anywhere, even at the trailhead.';
      }
      // Reset to step 1
      document.querySelectorAll('.onb-step').forEach(s => s.classList.remove('active'));
      document.querySelectorAll('.onb-dot').forEach(d => d.classList.remove('active'));
      document.getElementById('onb-step-1').classList.add('active');
      document.getElementById('onb-dot-1').classList.add('active');
      document.getElementById('onb-next-btn').textContent = 'Next';
      document.getElementById('onb-overlay').classList.add('open');
    }

    function nextOnboardingStep() {
      if (onbCurrentStep < ONB_TOTAL) {
        document.getElementById(`onb-step-${onbCurrentStep}`).classList.remove('active');
        document.getElementById(`onb-dot-${onbCurrentStep}`).classList.remove('active');
        onbCurrentStep++;
        document.getElementById(`onb-step-${onbCurrentStep}`).classList.add('active');
        document.getElementById(`onb-dot-${onbCurrentStep}`).classList.add('active');
        if (onbCurrentStep === ONB_TOTAL) {
          document.getElementById('onb-next-btn').textContent = 'Done';
        }
      } else {
        completeOnboarding();
      }
    }

    function completeOnboarding() {
      localStorage.setItem('rr_onboarded', '1');
      document.getElementById('onb-overlay').classList.remove('open');
    }
    ```
