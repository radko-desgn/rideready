<objective>
## Goal
Extend the sag verify modal with a PSI corrector: rider enters their measured sag, app calculates how much to add or remove.

## Purpose
The verify guide (05-01) tells riders how to measure sag but leaves them on their own once they have the number. This closes the loop — measured sag becomes an actionable PSI adjustment in the same modal, without switching screens.

## Output
- Sag modal extended: input field for measured sag + live PSI correction result
- `data-sag-travel-max` on the ? button so the corrector knows travel for forks
- `calcSagCorrection()` function for live calculation
- Forks: rider enters mm → app converts to % using travel, shows correction
- Shocks: rider enters % directly (no stroke data available) → same correction logic
</objective>

    **5. Add `calcSagCorrection()`**:
    ```js
    function calcSagCorrection() {
      const backdrop  = document.getElementById('sag-modal-backdrop');
      const sagMin    = parseFloat(backdrop.dataset.sagMin);
      const sagMax    = parseFloat(backdrop.dataset.sagMax);
      const travelMax = backdrop.dataset.sagTravelMax ? parseFloat(backdrop.dataset.sagTravelMax) : null;
      const input     = parseFloat(document.getElementById('sag-corrector-input').value);
      const result    = document.getElementById('sag-corrector-result');

      if (isNaN(input) || input <= 0) {
        result.className = 'sag-corrector-result';
        result.textContent = '';
        return;
      }

      const measuredPct = travelMax ? (input / travelMax) * 100 : input;
      const targetMid   = (sagMin + sagMax) / 2;
      const deviation   = measuredPct - targetMid;
      const correction  = Math.round(deviation);  // 1 PSI per 1% deviation

      if (measuredPct >= sagMin && measuredPct <= sagMax) {
        result.className = 'sag-corrector-result show good';
        result.textContent = '✓ Sag looks good';
      } else if (correction > 0) {
        result.className = 'sag-corrector-result show adjust';
        result.textContent = `↑ Add ${correction} PSI (sag is too deep)`;
      } else {
        result.className = 'sag-corrector-result show adjust';
        result.textContent = `↓ Remove ${Math.abs(correction)} PSI (sag is too firm)`;
      }
    }
    ```
